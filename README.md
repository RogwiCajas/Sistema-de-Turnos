# Sistema-de-Turnos

## **Descripción:**

Proyecto correspondiente al primer parcial de la materia "Estructura de Datos", el proyecto
busca simular la atencion a pacientes dentro de un hospital en base a la prioridad 
de los sintomas del paciente.
